# src/app/api — серверные эндпоинты

Route Handlers Next.js. Выполняются ТОЛЬКО на сервере.

- Здесь и только здесь используются секретные ключи (`ANTHROPIC_API_KEY` и т.п.).

- Клиентские компоненты обращаются к этим эндпоинтам по HTTP, а не к внешним сервисам напрямую.
