# src/app — страницы и роуты

Маршрутизация Next.js (App Router): страницы, layout'ы, серверные эндпоинты.

- Файлы `page.tsx` / `layout.tsx` — UI-роуты.
- Подпапка `api/` — серверные эндпоинты (Route Handlers). Только здесь живут секретные
  ключи и обращения к Claude / Supabase. Браузер не ходит во внешние сервисы напрямую.
