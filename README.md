# Life RPG

Приложение превращает реальные дела в RPG: вводишь действие — AI оценивает его сложность и пользу, начисляется опыт, растут характеристики и уровень. Дисциплина поощряется ежедневным стриком, а прогресс привязан к аккаунту.

## Возможности

- Ввод действий обычным текстом.
- AI-оценка сложности действия через Groq (модели Llama).
- Фильтр бессмысленных и негативных действий (не засчитываются).
- Опыт, уровни и «живые» некруглые значения XP.
- Характеристики: Интеллект, Сила, Творчество — растут от профильных действий.
- Дисциплина как ежедневный стрик (текущий и рекордный).
- Аккаунты и сохранение прогресса (Supabase, доступ к данным защищён через RLS).

## Стек

- **Next.js** (App Router) + **TypeScript**
- **Tailwind CSS**
- **Supabase** (Postgres + Auth)
- **Groq** (AI-оценка)
- Деплой на **Vercel**

## Локальный запуск

```bash
git clone https://github.com/FullvarFront/life-rpg.git
cd life-rpg
npm install
cp .env.example .env   # затем заполни ключи (см. ниже)
npm run dev
```

Приложение откроется на http://localhost:3000. SQL-схема БД — в [`supabase/schema.sql`](supabase/schema.sql) (выполнить в Supabase → SQL Editor).

## Переменные окружения

| Переменная | Описание |
|---|---|
| `GROQ_API_KEY` | Ключ Groq API для AI-оценки действий (только на сервере). |
| `NEXT_PUBLIC_SUPABASE_URL` | URL проекта Supabase. |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Публичный anon/publishable ключ Supabase (безопасен для клиента). |
