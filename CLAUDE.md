# Life RPG — правила проекта

## Что это
Веб-приложение, отображающее жизнь как RPG-игру. Пользователь вводит действие, AI (Claude) оценивает его сложность, начисляется опыт (XP).

## Стек
Next.js (App Router) + TypeScript (strict) + Tailwind CSS. БД: Supabase (Postgres). AI: Claude API.

## Архитектура — слои (не смешивать!)
- src/components/ — UI. Только отображение. Не содержит игровой логики и сетевых запросов.
- src/game/ — чистая игровая логика. Функции без побочных эффектов: на вход данные, на выход данные. Здесь правила XP, уровней, перевода сложности в опыт. Покрывается тестами в первую очередь.
- src/app/api/ — серверные эндпоинты. ТОЛЬКО здесь живут секретные ключи и обращения к Claude/Supabase. Браузер никогда не ходит в Claude напрямую.
- src/lib/ — клиенты внешних сервисов и утилиты.
- src/types/ — общие типы.

## Правила кода
- TypeScript strict. Никаких any — если тип неизвестен, спрашивай или используй unknown.
- Секретные ключи (ANTHROPIC_API_KEY и т.п.) используются ТОЛЬКО на сервере (src/app/api/), никогда в клиентских компонентах.
- Игровая логика (src/game/) не знает про React, БД и сеть. Чистые функции.
- Делай маленькие, понятные изменения. Если задача большая — предложи разбить.
- Пиши код в стиле окружающего кода.

## Команды
- npm run dev — запуск в разработке
- npm run build — сборка
- npm run lint — проверка линтером
